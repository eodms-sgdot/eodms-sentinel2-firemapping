The data for large fire polygons was downloaded from CFS. I have created a geopackage file from the downloaded shp file.
The original shape file and metadata documents can be downloaded from:
https://cwfis.cfs.nrcan.gc.ca/downloads/nfdb/fire_poly/current_version/
Ref: https://cwfis.cfs.nrcan.gc.ca/en/catalogue
